About CSS




Cascading Style Sheets is a style sheet language used for describing the presentation of a document written in a markup language such as HTML or XML. CSS is a cornerstone technology of the World Wide Web, alongside HTML and JavaScript.





How To Write

  1. To style a element 
    
                        
    element{
    }
  2. To set color of body red 
        
                        
    body{
        background-color:red;
    }
  3. To set color of h1 red 
            
                        
    h1{
      color:red;
    }
  4. To add animation to a div border for infinite 
                
                        
    div{
        border:2px solid black;
        animation:example 2s infinite;
    } 
    
    @keyframes example{
        from{border-color:black;}
        to{border-color:yellow;}
    }
  5. To style a id 
                    
                        
    #navbar{
    }
  6. To style a class 
                        
                        
    .box{
    }
  7. To giving padding and margin to an element 
                            
                        
    element{
       padding:10px;
       margin:10px;
    }
